# CareerSim AI — Next.js 15 Application

This is the production Next.js 15 + TypeScript + Tailwind CSS port of the CareerSim AI employment simulator prototype.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom design system CSS rules
- **Animations**: Framer Motion

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Open browser**:
   Go to [http://localhost:3000](http://localhost:3000) to view the application.

## Routing
- `/` — Landing Page
- `/careers` — Career Selection
- `/simulation` — Multi-Agent setup pipeline
- `/dashboard` — Student Dashboard
- `/mission` — Simulator workspace
- `/submission` — File submission workspace
- `/review` — AI Reviewer feedback summary
- `/passport` — Skill passport evidence checklist
- `/report` — Career roadmap and employability analyzer
